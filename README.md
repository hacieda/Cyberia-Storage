# Cyberia-Storage

**Cyberia Storage** - It is end-to-end encrypted file storage where even the server cannot know what you are storing. All data is encrypted on the client, and the keys never leave the device. 

The system stores only the minimum metadata required to authenticate users,
enforce storage quotas, and manage uploaded files. Activation history is not
stored separately in the database for the MVP. Public demo tokens can be reused,
while private tokens are marked as used after activation.
